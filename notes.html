Tags I've Learned and What They Do

  1. < em > < / em >
    Displays text in italics to add emphasis to words.
  2. < br >
    3. Creates space between sentences or headings. It also doesn't need a closing tag
  3. < abbr > < /abbr >
    4. An abbreviation tag that helps define abbreviationsor acronyms
  4. < sup > < / sup >
    5. Does superscript which can be used adding th , rd , st and nd
  5. < sub > < / sub >
    6. Does subscript which can be used for chemical formulas like H2O
